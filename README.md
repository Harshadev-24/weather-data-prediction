## Dataset
  **Description:**
  This dataset contains historical global weather records from 1750 to 1954, designed for climate and temperature analysis and prediction tasks in data science and machine learning projects.

  **Features/Columns:**
 - dt: Date of record (YYYY-MM-DD).
 - LandAverageTemperature: Average land temperature recorded (째C).
 - LandAverageTemperatureUncertainty: Uncertainty value for the above temperature.
 - LandMaxTemperature: Maximum land temperature measured (째C).
 - LandMaxTemperatureUncertainty: Uncertainty in max temperature.
 - LandMinTemperature: Minimum land temperature measured (째C).
 - LandMinTemperatureUncertainty: Uncertainty in min temperature.
 - LandAndOceanAverageTemperature: Average combined temperature of land and ocean (째C).
 - LandAndOceanAverageTemperatureUncertainty: Uncertainty in combined temperature.
